## 一、产品概述

### 1.1 产品定位

**"AI小商，你的智慧校园AI伙伴"**

AI小商是一款基于讯飞开放平台打造的智慧校园AI助手，通过自然语言对话、语音交互、多模态识别等AI能力，一站式解决师生的校园生活、学习、办公需求。

### 1.2 核心价值

| 用户 | 价值 |
|-----|-----|
| 学生 | 省时间：减少多系统切换；有个性：AI记住你的偏好 |
| 学校 | 减轻行政咨询压力；积累数字化经验 |
| 比赛 | 最大化讯飞API集成度；完整闭环展示 |

### 1.3 目标用户

**学生用户画像：**
- 大二学生，经济管理学院
- 每日困扰：课表记不住、开会笔记乱、选课迷茫
- 使用习惯：习惯语音输入，希望快速得到答案

**管理用户画像：**
- 教务处老师
- 维护校园FAQ知识库，查看学生使用数据

## 三、详细功能说明

### 3.1 登录注册

**流程：** 微信扫码/手机号 → 完善个人信息（专业/年级/目标）→ 进入首页

**字段：** nickname, openid, phone, major, grade, goal

### 3.2 智能课表管家

**F2-1 课表图片导入（OCR）**
```
用户上传课表图片
    ↓
讯飞OCR识别 → 星火大模型解析 → 用户确认 → 写入数据库
```

**F2-2 自然语言添加日程**
```
"周二上午9点去9教开会"
    ↓
星火解析意图 → 提取：时间、地点、事项 → 写入数据库
```

**F2-3 首页今日提醒**
```
今明两日课程醒目展示 + AI学习建议
```

### 3.3 课程·会议回顾总结

**F3-1 录音转写**
```
上传MP3（≤50MB）→ 选择类型（课程/会议）→ 讯飞ASR转文字
```

**F3-2 AI生成总结**
```
转写文字 → 星火大模型 → 结构化总结
    课程：主题、核心知识点、重点难点、金句、预习建议
    会议：主题、讨论要点、决议事项、待办事项
```

**F3-3 导出DOCX/PPT**
```
总结 → python-docx → DOCX文件下载
课程总结 → python-pptx → PPT文件下载
```

### 3.4 AI学姐对话

**F6-1 校园专属问答**
- RAG知识库检索 + 星火大模型生成回答
- 联动前5个功能的数据（课程、日程、回顾记录等）

**F6-2 语音交互**
- 讯飞ASR：语音输入 → 识别为文字
- 讯飞TTS：回复文字 → 合成语音播放

**F6-3 情感陪伴模式**
- 点击❤️切换为温暖倾听模式
- 讯飞NLP情感分析识别负面情绪
- 自动调整回复语气和内容

## 四、智能翻译

### 4.1 即时翻译

**功能描述：** 用户在左侧输入文本，选择目标语言，点击翻译按钮后，右侧实时输出翻译结果。

**页面布局：**
```
┌─────────────────────────────────────────────────────────────────┐
│  即时翻译                                                         │
├─────────────────────┬─────────────┬─────────────────────────────┤
│      输入区域        │  语言选择   │        输出区域              │
│  (可输入/粘贴文本)   │  ▼ 中文     │     (翻译结果)               │
│                     │  ▼ 英文     │                              │
│                     │  ▼ 日文     │                              │
│                     │  ▼ 韩文     │                              │
│                     │  ▼ 法文     │                              │
│                     │  ▼ 德文     │                              │
├─────────────────────┴─────────────┴─────────────────────────────┤
│                              [翻译]                               │
└─────────────────────────────────────────────────────────────────┘
```

**支持语言：** 中文、英文、日文、韩文、法文、德文、西班牙文、俄文、阿拉伯文等

**交互流程：**
```
用户输入文本 → 选择目标语言 → 点击翻译 → 调用星火大模型翻译API → 显示结果
```

**技术实现：**
- 使用讯飞星火大模型的文本翻译能力
- 支持多语言互译
- 翻译结果可一键复制

### 4.2 文档翻译

**功能描述：** 用户上传txt或docx文档，选择目标语言，系统翻译后支持预览和Word导出。

**页面布局：**
```
┌─────────────────────────────────────────────────────────────────┐
│  文档翻译                                                         │
├─────────────────────────────────────────────────────────────────┤
│  ┌───────────────────────────────────────────────────────────┐  │
│  │  📁 点击或拖拽文件到此处上传                               │  │
│  │     支持 .txt, .docx 文件                                  │  │
│  └───────────────────────────────────────────────────────────┘  │
│                                                                  │
│  目标语言：▼ 中文                                                │
│                                                                  │
│  [                翻译文档                ]                      │
│                                                                  │
│  ─────────────────────────────────────────────────────────────  │
│  翻译预览                                                         │
│  ┌───────────────────────────────────────────────────────────┐  │
│  │                                                          │  │
│  │  （显示翻译后的文档内容）                                  │  │
│  │                                                          │  │
│  └───────────────────────────────────────────────────────────┘  │
│                                                                  │
│  [下载Word文档]                                                  │
└─────────────────────────────────────────────────────────────────┘
```

**交互流程：**
```
用户上传文档(txt/docx) → 选择目标语言 → 点击翻译 →
后端解析文档内容 → 调用星火大模型翻译 → 返回翻译结果 →
前端展示预览 → 用户可导出为Word文档
```

**技术实现：**
1. 文档解析：
   - .txt 文件：直接读取文本内容
   - .docx 文件：使用 python-docx 库解析段落
2. 翻译处理：分段落调用星火翻译API，支持进度显示
3. 结果展示：前端渲染翻译后的文本
4. 导出功能：使用 python-docx 生成Word文档

**状态流转：**
```
空闲 → 上传中 → 翻译中(显示进度%) → 完成 → 可预览/导出
                                         ↓
                                    翻译失败 → 显示错误信息
```

## 五、校园活动助手

### 5.1 功能概述

**定位**：帮助学生快速生成活动策划方案和宣传文案，支持一键发送到飞书群。

**核心价值**：
- 节省策划时间：5分钟生成完整方案

- 便捷传播：一键推送到飞书群

### 5.2 活动策划生成

**功能描述**：用户选择活动类型、填写基本信息，AI生成完整的活动策划方案。

**输入表单**：
| 字段 | 类型 | 说明 |
|-----|------|------|
| 活动类型 | 下拉选择 | 文艺类、体育类、学术类、志愿类、团建类、其他 |
| 活动主题 | 文本输入 | 例如"校园歌手大赛" |
| 预计人数 | 下拉选择 | 小规模(≤50)、中规模(50-200)、大规模(≥200) |
| 预算范围 | 下拉选择 | 低(<500元)、中(500-2000元)、高(>2000元) |
| 活动时间 | 日期选择 | 初步意向日期 |
| 特殊需求 | 多选框 | 需要抽奖、需要嘉宾、室外活动、线上活动 |

**AI输出策划方案**：
```
【活动名称】：校园歌手大赛
【活动概述】：面向全校学生的歌唱比赛，发掘校园歌王

【时间安排】
  - 前期准备：提前2周开始宣传，报名截止前1周
  - 活动当天：海选(上午) → 复赛(下午) → 决赛(晚上)
  - 后期收尾：奖品发放、新闻稿发布

【人员分工】
  - 策划组：整体统筹、时间把控
  - 宣传组：海报制作、公众号推文、抖音短视频
  - 后勤组：场地布置、音响设备、物资采购
  - 机动组：现场秩序、突发情况处理

【预算清单】
  - 奖品：一等奖500元、二等奖300元、三等奖100元
  - 物料：海报50张、背景板1块、话筒电池等
  - 预计总支出：约1200元

【风险预案】
  - 下雨：启用室内备选场地（九教报告厅）
  - 设备故障：备用音响+紧急联系人

【宣传方案】
  - 线上：公众号推文、抖音挑战赛、朋友圈集赞
  - 线下：海报张贴、传单发放、广播站播报
```

**页面布局**：

### 5.5 技术实现

**后端接口**：
| 方法 | 路径 | 功能 |
|-----|------|------|
| POST | `/api/activity/generate-plan` | 生成活动策划方案 |
| POST | `/api/activity/generate-copy` | 生成宣传文案 |
| POST | `/api/activity/send-feishu` | 发送到飞书群 |

**技术要点**：
- 复用星火大模型生成策划和文案
- 飞书使用预配置Webhook，无用户配置成本
- post消息格式支持加粗、换行、emoji
