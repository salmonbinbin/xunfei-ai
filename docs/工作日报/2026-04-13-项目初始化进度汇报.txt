六、启动命令
================================================================================

【前端启动】
  cd frontend
  npm install
  npm run dev
  # 访问 http://localhost:5173

【后端启动】
  cd backend
  python3 -m venv venv
  source venv/bin/activate
  pip install -r requirements.txt
  # 配置 .env 文件（复制 .env.example 为 .env 并填入配置）
  uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
  # 访问 http://localhost:8000/docs (API文档)
