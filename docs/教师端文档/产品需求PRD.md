# AI小商 - 教师端产品需求文档（PRD）

> 产品愿景、功能需求、用户体验设计。

---

## 一、产品概述

### 1.1 产品定位

**"AI小商教师端，智能教学好帮手"**

AI小商教师端是面向广州商学院教师的智能教学助手，通过AI能力辅助教师完成成绩管理、通知发布、备课教案生成等工作，提升教学效率。

### 1.2 核心价值

| 价值点 | 说明 |
|-------|------|
| 省时 | AI自动处理繁琐的成绩统计、通知格式等工作 |
| 智能 | 基于大模型理解教师意图，生成高质量教学内容 |
| 高效 | 一键生成可视化图表、规范通知、教学PPT |

### 1.3 目标用户

**教师用户画像：**
- 专业课教师（经济管理、计算机等）
- 每周需要处理成绩、发布通知、备课
- 希望减少重复性行政工作

## 三、详细功能说明

### 3.1 登录注册

**流程：** 手机号注册 → 验证身份（教师）→ 进入教师端

**字段：** phone, password, name, department, course (教授课程)

**权限控制：**
- 教师角色（role=teacher）只能访问教师端
- 学生角色（role=student）只能访问学生端

### 3.2 智能成绩管理

**F2-1 成绩表上传**
```
教师上传Excel成绩表（.xlsx格式）
    ↓
后端解析Excel（课程名、学生姓名、各项成绩）
    ↓
显示成绩预览表格
```

**成绩表格式要求：**
| 列名 | 说明 | 必填 |
|-----|------|-----|
| 学生姓名 | 文本 | 是 |
| 学号 | 文本 | 否 |
| 平时分 | 数字 | 是 |
| 期中分 | 数字 | 否 |
| 期末分 | 数字 | 是 |
| 实验/实践分 | 数字 | 否 |

**F2-2 AI自动计算**
```
用户配置成绩权重（如：平时30% 期中20% 期末50%）
    ↓
AI自动计算：平时分、期末分、总评（加权平均）
    ↓
AI自动计算：排名（按总评降序）
    ↓
生成成绩统计表
```

**计算规则：**
- 总评 = 平时分 × 权重1 + 期中分 × 权重2 + 期末分 × 权重3 + 实验分 × 权重4
- 权重默认：平时40% 期中20% 期末40%
- 缺考/缓考标记为"缓考"/"缺考"，不参与计算

**F2-3 可视化图表**
```
成绩分布直方图（按区间：<60, 60-70, 70-80, 80-90, >90）
    ↓
班级对比柱状图（同课程不同班级对比）
    ↓
成绩趋势折线图（如果有多次考试数据）
    ↓
导出图表为PNG图片
```

**F2-4 导出功能**
- 导出完整成绩表为Excel
- 导出统计图表为PNG

### 3.3 智能通知发布

**F3-1 通知生成**
```
教师输入通知主题/关键词（如："关于期末考试安排"）
    ↓
AI理解意图，自动生成：
  - 标题（规范格式）
  - 正文（正式语气，格式规范）
  - 截止时间（根据内容推断）
  - 落款（教师姓名+日期）
```

**生成的通知格式示例：**
```
【通知】
各位同学：

关于2026年春季学期期末考试安排的通知

一、考试时间
2026年6月15日（周一）上午9:00-11:00

二、考试地点
九教501、502、503教室

三、注意事项
1. 请提前15分钟到达考场
2. 携带学生证参加考试
3. 关闭手机并放在指定位置

四、联系人
张老师（办公室：行政楼302）

广州商学院经济管理学院
2026年5月10日
```

**F3-2 通知编辑**
- 教师可修改生成的标题、正文
- 支持富文本编辑（加粗、换行）
- 一键复制到剪贴板

**F3-3 通知发布**
- 复制通知文本
- 发送到飞书群（复用学生端飞书推送能力）

### 3.4 智能备课教案生成

**F4-1 知识点输入**
```
教师输入章节标题/知识点（如："市场营销第四章-消费者行为分析"）
    ↓
AI理解后自动补充：
  - 教学目标
  - 教学内容框架
  - 重点难点
  - 教学方法建议
```

**F4-2 教学大纲生成**
```
AI生成结构化教学大纲：
  - 章节概述
  - 教学目标（知识、技能、情感）
  - 教学内容（分小节）
  - 重难点分析
  - 课时安排（建议2-4学时）
  - 思考题/作业
```

**F4-3 PPT自动生成**
```
基于教学大纲，AI生成PPT：
  - 封面页（课程名+章节+教师名）
  - 目录页
  - 内容页（每小节1-3页）
  - 总结页
  - 作业页
    ↓
使用python-pptx生成.pptx文件下载
```

## 六、技术约束

- 成绩Excel文件大小限制：≤5MB
- 单次生成PPT页数限制：≤50页
- 教师注册需验证手机号格式
- 所有AI生成内容需教师确认后方可使用
