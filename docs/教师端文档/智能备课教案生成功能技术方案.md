# 智能备课教案生成功能技术方案

> 本文档详细描述教师端"智能备课教案生成"功能的技术实现方案。
> 使用讯飞开放平台"智能PPT生成"API实现。

---

## 一、功能概述

### 1.1 功能描述

智能备课教案生成是教师端的第四个核心功能（P1优先级），帮助教师快速生成教学大纲和PPT演示文稿。

**输入**：教师输入知识点、课程名称、授课对象等信息
**输出**：AI生成的教学大纲 + 可下载的PPT文件

### 1.2 用户交互流程

```
┌─────────────┐     ┌──────────────┐     ┌─────────────┐     ┌──────────────┐
│  教师输入   │ ──▶ │  AI生成大纲   │ ──▶ │  教师确认   │ ──▶ │  生成PPT    │
│  课程信息   │     │  （可调整）   │     │  大纲       │     │  下载        │
└─────────────┘     └──────────────┘     └─────────────┘     └──────────────┘
```

**两种模式**：
1. **快速模式**：直接输入要求，AI一次性生成大纲+PPT
2. **分步模式**：先生成大纲 → 教师确认/修改 → 再生成PPT

## 二、讯飞PPT生成API

### 2.1 API接口总览

| 接口 | 方法 | 地址 | 用途 |
|-----|------|------|------|
| 主题列表查询 | POST | `/v2/aippt/theme/list` | 获取可选PPT模板 |
| PPT直接生成 | POST | `/v2/aippt/create` | 输入要求，直接生成PPT |
| 大纲生成 | POST | `/v2/aippt/createOutline` | 只生成大纲，不生成PPT |
| 通过大纲生成PPT | POST | `/v2/aippt/createPPTByOutline` | 基于已有大纲生成PPT |
| PPT进度查询 | POST | `/v2/aippt/process` | 查询PPT生成状态 |

### 2.2 鉴权方式

与星火大模型API一致，使用签名认证：

```
HTTP Request Header:
  appId: 应用ID
  timestamp: 时间戳（秒，与服务器相差5分钟内有效）
  signature: 签名
```

### 2.3 签名算法

```python
import hashlib
import hmac
import base64

def generate_signature(app_id: str, api_key: str, timestamp: str) -> str:
    """
    生成讯飞API签名
    签名规则：HMAC-SHA256(appId + timestamp, apiKey)，然后Base64编码
    """
    sign_str = f"{app_id}{timestamp}"
    signature = hmac.new(
        api_key.encode('utf-8'),
        sign_str.encode('utf-8'),
        digestmod=hashlib.sha256
    ).digest()
    return base64.b64encode(signature).decode('utf-8')
```

# app/services/lesson_plan_service.py

"""
智能备课教案服务 - 基于讯飞PPT生成API
"""
import asyncio
import json
import logging
import time
from typing import Dict, Any, Optional, List
from datetime import datetime

import httpx
from app.config import settings
from app.services.xinghuo_service import xinghuo_service

logger = logging.getLogger("lesson_plan_service")


class LessonPlanService:
    """教案服务类"""

    # 讯飞PPT API配置
    PPT_API_BASE = "https://api.xf-yun.com"
    PPT_THEME_LIST_URL = f"{PPT_API_BASE}/v2/aippt/theme/list"
    PPT_CREATE_URL = f"{PPT_API_BASE}/v2/aippt/create"
    PPT_CREATE_OUTLINE_URL = f"{PPT_API_BASE}/v2/aippt/createOutline"
    PPT_CREATE_BY_OUTLINE_URL = f"{PPT_API_BASE}/v2/aippt/createPPTByOutline"
    PPT_PROCESS_URL = f"{PPT_API_BASE}/v2/aippt/process"

    # 轮询配置
    MAX_POLL_ATTEMPTS = 60  # 最多轮询60次
    POLL_INTERVAL = 3  # 每3秒轮询一次

    def __init__(self):
        self.app_id = settings.XFYUN_APP_ID
        self.api_key = settings.XFYUN_API_KEY
        self.api_secret = settings.XFYUN_API_SECRET
        self.logger = logging.getLogger("lesson_plan_service")

    def _generate_signature(self, timestamp: str) -> str:
        """生成API签名"""
        import hashlib
        import hmac
        import base64

        sign_str = f"{self.app_id}{timestamp}"
        signature = hmac.new(
            self.api_key.encode('utf-8'),
            sign_str.encode('utf-8'),
            digestmod=hashlib.sha256
        ).digest()
        return base64.b64encode(signature).decode('utf-8')

    def _build_headers(self, timestamp: str) -> Dict[str, str]:
        """构建请求头"""
        return {
            "Content-Type": "application/json",
            "appId": self.app_id,
            "timestamp": timestamp,
            "signature": self._generate_signature(timestamp)
        }

    async def _http_post(self, url: str, data: Dict[str, Any]) -> Dict[str, Any]:
        """HTTP POST请求（带重试）"""
        timestamp = str(int(time.time()))
        headers = self._build_headers(timestamp)

        async with httpx.AsyncClient(timeout=120.0) as client:
            for attempt in range(3):
                try:
                    self.logger.info(f"[HTTP] POST {url} (attempt {attempt + 1}/3)")
                    response = await client.post(url, headers=headers, json=data)
                    result = response.json()

                    # 检查讯飞返回的错误码
                    if result.get("code") == 20007:
                        self.logger.error(f"[HTTP] 鉴权错误: {result}")
                        raise Exception(f"讯飞API鉴权失败: {result.get('desc')}")

                    self.logger.info(f"[HTTP] Response: code={result.get('code')}, desc={result.get('desc')}")
                    return result

                except httpx.TimeoutException:
                    self.logger.warning(f"[HTTP] Timeout (attempt {attempt + 1}/3)")
                    if attempt == 2:
                        raise Exception("讯飞API请求超时")
                except Exception as e:
                    self.logger.error(f"[HTTP] Error: {str(e)}")
                    if attempt == 2:
                        raise
                    await asyncio.sleep(1 * (attempt + 1))

    async def get_ppt_themes(
        self,
        style: Optional[str] = None,
        color: Optional[str] = None,
        industry: Optional[str] = None,
        page: int = 1,
        page_size: int = 10
    ) -> Dict[str, Any]:
        """
        获取PPT主题列表

        Args:
            style: 风格类型（简约/卡通/商务/创意/国风/清新/扁平/插画/节日）
            color: 颜色类型（蓝色/绿色/红色/紫色/黑色/灰色/黄色/粉色/橙色）
            industry: 行业类型
            page: 页码
            page_size: 每页数量

        Returns:
            {"total": int, "records": [...]}
        """
        self.logger.info(f"[LessonPlan] get_ppt_themes: style={style}, color={color}, industry={industry}")

        data = {
            "pageNum": page,
            "pageSize": page_size
        }
        if style:
            data["style"] = style
        if color:
            data["color"] = color
        if industry:
            data["industry"] = industry

        result = await self._http_post(self.PPT_THEME_LIST_URL, data)

        if not result.get("flag"):
            raise Exception(f"获取PPT主题失败: {result.get('desc')}")

        return {
            "total": result.get("data", {}).get("total", 0),
            "records": result.get("data", {}).get("records", [])
        }

    async def generate_outline(
        self,
        query: str,
        language: str = "cn",
        search: bool = False
    ) -> Dict[str, Any]:
        """
        生成PPT大纲

        Args:
            query: 生成要求（最多12000字）
            language: 语种（cn/en/ja...）
            search: 是否联网搜索

        Returns:
            {"sid": str, "outline": {"title": str, "subTitle": str, "chapters": [...]}}
        """
        self.logger.info(f"[LessonPlan] generate_outline: query长度={len(query)}")

        data = {
            "query": query,
            "language": language,
            "search": search
        }

        result = await self._http_post(self.PPT_CREATE_OUTLINE_URL, data)

        if not result.get("flag"):
            code = result.get("code")
            if code == 20005:
                raise Exception("大纲生成失败，请检查是否存在敏感词汇")
            raise Exception(f"大纲生成失败: {result.get('desc')}")

        outline_data = result.get("data", {})
        self.logger.info(f"[LessonPlan] 大纲生成成功: sid={outline_data.get('sid')}")

        return {
            "sid": outline_data.get("sid"),
            "outline": outline_data.get("outline")
        }

    async def generate_ppt_by_outline(
        self,
        outline: Dict[str, Any],
        query: str,
        template_id: Optional[str] = None,
        author: str = "AI小商",
        language: str = "cn",
        search: bool = False,
        is_figure: bool = False,
        ai_image: str = "normal",
        is_card_note: bool = False,
        outline_sid: Optional[str] = None
    ) -> Dict[str, Any]:
        """
        通过大纲生成PPT

        Args:
            outline: 大纲数据（JSON对象）
            query: 生成要求
            template_id: 模板ID
            author: PPT作者
            language: 语种
            search: 是否联网搜索
            is_figure: 是否AI配图
            ai_image: AI配图类型（normal/advanced）
            is_card_note: 是否生成演讲备注
            outline_sid: 大纲SID（用于关联）

        Returns:
            {"sid": str, "title": str, "ppt_url": str, ...}
        """
        self.logger.info(f"[LessonPlan] generate_ppt_by_outline: outline有{len(outline.get('chapters', []))}章节")

        data = {
            "outline": outline,
            "query": query,
            "author": author,
            "language": language,
            "search": search,
            "isFigure": is_figure,
            "aiImage": ai_image,
            "isCardNote": is_card_note
        }
        if template_id:
            data["templateId"] = template_id
        if outline_sid:
            data["outlineSid"] = outline_sid

        result = await self._http_post(self.PPT_CREATE_BY_OUTLINE_URL, data)

        if not result.get("flag"):
            code = result.get("code")
            if code == 20006:
                raise Exception("PPT生成失败，请稍后重试或联系技术人员")
            raise Exception(f"PPT生成失败: {result.get('desc')}")

        ppt_data = result.get("data", {})
        self.logger.info(f"[LessonPlan] PPT生成任务已创建: sid={ppt_data.get('sid')}")

        return {
            "sid": ppt_data.get("sid"),
            "cover_img_src": ppt_data.get("coverImgSrc"),
            "title": ppt_data.get("title"),
            "sub_title": ppt_data.get("subTitle"),
            "outline": ppt_data.get("outline")
        }

    async def poll_ppt_status(self, sid: str) -> Dict[str, Any]:
        """
        轮询PPT生成状态

        Args:
            sid: PPT任务ID

        Returns:
            {"ppt_status": str, "ppt_url": str, "total_pages": int, "done_pages": int}
        """
        self.logger.info(f"[LessonPlan] poll_ppt_status: sid={sid}")

        data = {"sid": sid}

        for attempt in range(self.MAX_POLL_ATTEMPTS):
            result = await self._http_post(self.PPT_PROCESS_URL, data)

            if not result.get("flag"):
                raise Exception(f"查询PPT状态失败: {result.get('desc')}")

            ppt_data = result.get("data", {})
            ppt_status = ppt_data.get("pptStatus")

            self.logger.info(f"[LessonPlan] 状态: {ppt_status}, 完成页数: {ppt_data.get('donePages')}/{ppt_data.get('totalPages')}")

            if ppt_status == "done":
                return {
                    "ppt_status": "completed",
                    "ppt_url": ppt_data.get("pptUrl"),
                    "ai_image_status": ppt_data.get("aiImageStatus"),
                    "card_note_status": ppt_data.get("cardNoteStatus"),
                    "total_pages": ppt_data.get("totalPages"),
                    "done_pages": ppt_data.get("donePages")
                }
            elif ppt_status == "build_failed":
                raise Exception(f"PPT生成失败: {ppt_data.get('errMsg')}")

            # 等待下次轮询
            await asyncio.sleep(self.POLL_INTERVAL)

        raise Exception("PPT生成超时，请稍后重试")

    def build_lesson_plan_prompt(
        self,
        title: str,
        knowledge_points: str,
        course_name: Optional[str] = None,
        target_audience: Optional[str] = None,
        teaching_hours: Optional[int] = None
    ) -> str:
        """
        构建教案生成Prompt

        Args:
            title: 教案标题
            knowledge_points: 知识点
            course_name: 课程名称
            target_audience: 授课对象
            teaching_hours: 课时数

        Returns:
            生成大纲用的prompt字符串
        """
        prompt = f"""请为以下教学内容生成一份详细的教学大纲：

## 教案标题
{title}

## 课程名称
{course_name or "未指定"}

## 授课对象
{target_audience or "广州商学院学生"}

## 课时数
{teaching_hours or 2}课时

## 核心知识点
{knowledge_points}

请生成一份结构清晰、适合教学的大纲，包含：
1. 封面信息（标题、副标题）
2. 章节结构（每个章节包含章节标题和详细内容要点）
3. 每个章节的内容应该包含3-5个要点

要求：
- 适合高等教育课堂教学
- 内容专业、准确
- 结构层次分明
- 突出重难点"""
        return prompt

## 六、错误处理

### 6.1 业务错误码

| 错误码 | 描述 | 处理方式 |
|-------|------|---------|
| 20002 | 参数错误 | 检查输入参数 |
| 20005 | 大纲生成失败 | 可能含敏感词，建议用户修改知识点描述 |
| 20006 | PPT生成失败 | 建议重新生成或联系技术支持 |
| 20007 | 鉴权错误 | 检查讯飞API配置 |
| 9999 | 系统异常 | 重试或联系技术支持 |

### 6.2 重试机制

| 接口 | 最大重试 | 间隔递增 |
|-----|---------|---------|
| 主题列表查询 | 3次 | 1s, 2s, 3s |
| 大纲生成 | 3次 | 2s, 4s, 6s |
| PPT生成 | 3次 | 2s, 4s, 6s |
| 状态查询 | 不重试 | 轮询方式 |
