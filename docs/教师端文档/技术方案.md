## 二、数据库设计

### 2.1 扩展现有User表

```sql
-- 在现有users表中添加 role 字段
ALTER TABLE users ADD COLUMN role ENUM('student', 'teacher', 'admin') DEFAULT 'student';

-- 添加教师扩展信息表
CREATE TABLE teacher_profiles (
    id INT PRIMARY KEY AUTO_INCREMENT,
    user_id INT NOT NULL,
    department VARCHAR(100) COMMENT '院系',
    office VARCHAR(100) COMMENT '办公室',
    title VARCHAR(50) COMMENT '职称（教授/副教授/讲师）',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id)
);
```

### 2.2 新增成绩管理相关表

```sql
-- 成绩记录表
CREATE TABLE grade_records (
    id INT PRIMARY KEY AUTO_INCREMENT,
    teacher_id INT NOT NULL COMMENT '教师用户ID',
    course_name VARCHAR(200) NOT NULL COMMENT '课程名称',
    semester VARCHAR(50) COMMENT '学期（如2026春季）',
    class_name VARCHAR(100) COMMENT '班级名称',
    file_path VARCHAR(500) COMMENT '原始Excel路径',
    weights JSON COMMENT '成绩权重配置',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (teacher_id) REFERENCES users(id)
);

-- 成绩明细表
CREATE TABLE grade_items (
    id INT PRIMARY KEY AUTO_INCREMENT,
    record_id INT NOT NULL,
    student_name VARCHAR(100) NOT NULL,
    student_no VARCHAR(50),
    usual_score DECIMAL(5,2) COMMENT '平时分',
    midterm_score DECIMAL(5,2) COMMENT '期中分',
    final_score DECIMAL(5,2) COMMENT '期末分',
    practice_score DECIMAL(5,2) COMMENT '实验/实践分',
    total_score DECIMAL(5,2) COMMENT '总评',
    rank INT COMMENT '排名',
    status ENUM('normal', 'absent', 'deferred') DEFAULT 'normal' COMMENT '考试状态',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (record_id) REFERENCES grade_records(id)
);

-- 教案表
CREATE TABLE lesson_plans (
    id INT PRIMARY KEY AUTO_INCREMENT,
    teacher_id INT NOT NULL,
    title VARCHAR(200) NOT NULL COMMENT '章节标题',
    course_name VARCHAR(200) COMMENT '课程名称',
    outline TEXT COMMENT '教学大纲JSON',
    ppt_path VARCHAR(500) COMMENT '生成的PPT路径',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (teacher_id) REFERENCES users(id)
);

-- 通知记录表
CREATE TABLE notifications (
    id INT PRIMARY KEY AUTO_INCREMENT,
    teacher_id INT NOT NULL,
    title VARCHAR(200),
    content TEXT,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (teacher_id) REFERENCES users(id)
);
```

    │       └── grade_items (1:N)

## 八、权限控制

### 8.1 中间件验证

```python
# backend/app/utils/auth.py 新增

def get_current_teacher(current_user: User = Depends(get_current_user)):
    """验证是否为教师角色"""
    if current_user.role != 'teacher':
        raise HTTPException(status_code=403, detail="教师权限required")
    return current_user
```

### 8.2 前端路由守卫

```javascript
// router/index.js
router.beforeEach((to, from, next) => {
  const userStore = useUserStore()

  if (to.meta.requiresAuth && !userStore.isLoggedIn) {
    next('/login')
  }

  if (to.meta.role && userStore.user?.role !== to.meta.role) {
    next('/')  // 角色不匹配，跳转首页
  }

  next()
})
```
