**功能：**
- 日志列表（分页、筛选）

## 快速开始

### 1. 数据库初始化

```bash
# 执行数据库初始化脚本
mysql -u root -p ai_xiaoshang < scripts/init_admin_tables.sql
```

### 2. 启动后端

```bash
cd backend
uvicorn app.main:app --reload --port 8000
```

### 3. 启动前端

```bash
cd frontend
npm run dev
```

### 4. 访问管理端

```
http://localhost:5173/admin/login
```

**默认管理员账号：**
- 用户名：`admin`
- 密码：`123456`

## 开发规范

### 命名规范

- 路由命名：`/api/admin/xxx`
- 文件命名：`admin_xxx.js` / `AdminXXX.vue`
- 变量命名：camelCase
- 常量命名：UPPER_SNAKE_CASE

### 代码风格

- 使用 ESLint + Prettier
- Vue组件使用 `<script setup>`
- API统一封装在 `api/admin/` 目录
- 状态统一使用 Pinia

### 组件规范

- 使用 Element Plus 组件
- 自定义组件添加 `ElIcon` 图标
- 页面布局最大宽度 `1400px`
- 卡片圆角 `16px`
