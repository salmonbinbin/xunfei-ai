# AI小商 - "我的"模块设计文档

> 本文档定义"我的"个人中心模块的功能设计、交互规范、数据结构。
> 该模块与"选课助手"共用学生画像数据，尤其是发展方向（goal）字段。

---

## 一、模块概述

### 1.1 功能定位

"我的"是用户的个人主页，展示个人基本信息、学习数据统计和最近活动时间线，提供资料编辑和发展方向设置入口。

### 1.2 核心价值

- **个人信息管理**：修改昵称、专业、年级、发展目标
- **数据统计概览**：快速了解学习数据
- **最近活动入口**：快速跳转录音回顾、AI对话等
- **与选课助手联动**：发展方向（goal）影响选课推荐结果

## 三、功能详细设计

### 3.1 个人信心卡片

#### 信息展示
| 字段 | 来源 | 说明 |
|-----|------|------|
| 昵称 | `user.nickname` | 用户昵称，未设置显示"未设置昵称" |
| 学校 | 固定"广州商学院" | 固定显示 |
| 专业 | `student_profile.major` | 未设置显示"未设置专业" |
| 年级 | `student_profile.grade` | 显示"大{X}年级"，未设置显示"未设置" |
| 发展方向 | `student_profile.goal` | 可选：考研/考公/就业/出国/未定 |

#### 编辑资料按钮
- 点击弹出 `编辑资料` 对话框

#### 发展方向选择器
- 使用 `el-select` 下拉选择
- 选项：`考研`、`考公`、`就业`、`出国`、`未定`
- **选中后自动保存**，调用 `POST /api/auth/profile` 接口
- 修改后**同步影响选课助手**的推荐结果

---

### 3.2 编辑资料弹窗

#### 表单字段
| 字段 | 组件 | 说明 |
|-----|------|------|
| 昵称 | `el-input` | 最大20字符 |
| 专业 | `el-input` | 最大100字符 |
| 年级 | `el-select` | 选项：1、2、3、4 |
| 发展方向 | `el-select` | 选项：考研/考公/就业/出国/未定 |

#### 交互流程
```
点击[编辑资料]
    ↓
弹出编辑资料对话框（显示当前信息）
    ↓
用户修改表单
    ↓
点击[保存]
    ↓
调用 POST /api/auth/profile
    ↓
成功后：
    1. 关闭对话框
    2. 刷新页面显示新数据
    3. 若goal变更，同步更新userStore
```

#### 验证规则
| 字段 | 规则 |
|-----|------|
| 昵称 | 非空，最大20字符 |
| 专业 | 可空，最大100字符 |
| 年级 | 1-4整数 |
| 发展方向 | 必选，枚举值之一 |

---

### 3.3 数据统计（4宫格）

#### 指标卡片
| 指标 | 数据来源 | 说明 |
|-----|---------|------|
| 我的课程 | 课表数据 | 用户已导入的课程数量 |
| 录音回顾 | `GET /api/review` | 已完成的录音回顾数量 |
| AI对话 | `GET /api/chat/conversations` | 独立会话数量 |
| 日程记录 | `GET /api/schedule` | 日程总数量 |

#### 视觉规范
- 卡片采用 `border-radius: 16px`
- 图标背景使用半透明主色
- 数字使用 `font-size: 24px`，加粗
- 标签使用 `font-size: 12px`，灰色

---

### 3.4 最近活动时间线

#### 数据来源
| 活动类型 | API | 数量 |
|---------|-----|------|
| 录音回顾 | `GET /api/review` + `status=completed` | 取最新3条 |
| AI对话 | `GET /api/chat/conversations` | 取最新3条 |
| 日程 | `GET /api/schedule` | 取最新3条 |

#### 展示信息
| 字段 | 录音回顾 | AI对话 | 日程 |
|-----|---------|--------|------|
| 图标 | 🎙️ | 💬 | 📅 |
| 时间 | 相对时间（2小时前/昨天） | 同左 | 同左 |
| 标题 | 录音标题/章节名 | 会话标题 | 日程内容 |
| 副标题 | 时长 + 状态 | 消息数 | 时间 + 地点 |
| 点击行为 | 跳转 `/review/{id}` | 跳转 `/ai-sister?conv={id}` | 跳转 `/schedule` |

#### 时间格式化
```javascript
// 相对时间显示规则
< 1小时：XX分钟前
< 24小时：XX小时前
< 48小时：昨天
< 7天：X天前
>= 7天：月日（如 04-20）
```

#### 空状态
若无任何活动，显示：
```
暂无最近活动
开始使用AI小商，开启你的智慧校园生活吧
```

## 四、与选课助手的联动

### 4.1 数据同步机制

```
┌──────────────┐      修改goal       ┌──────────────┐
│  我的页面    │ ─────────────────► │  userStore   │
│              │                     │              │
│  发展方向    │                     │  profile.goal│
│  选择器      │                     │              │
└──────────────┘                     └──────┬───────┘
                                           │
                                           │ 下次访问时读取
                                           ▼
                                    ┌──────────────┐
                                    │  选课助手    │
                                    │              │
                                    │  影响推荐    │
                                    │  结果       │
                                    └──────────────┘
```

### 4.2 实现要点

1. **前端保存后同步store**：
```javascript
async function onGoalChange(newGoal) {
  await userStore.updateProfile({ goal: newGoal })
  // userStore.userInfo.profile.goal 已更新
  // 选课助手下次加载时会读取新值
}
```

2. **选课助手读取goal**：
```javascript
// CourseAdvisor.vue
const profileData = {
  major: store.profile.major,
  grade: store.profile.grade,
  goal: store.profile.goal  // 从userStore读取
}
```

3. **目标枚举一致性**：
```javascript
// 两个模块使用相同的枚举值
const GOALS = ['考研', '考公', '就业', '出国', '未定']
```

## 五、数据结构

### 5.1 前端状态（userStore）

```javascript
// stores/user.js
{
  userInfo: {
    id: number,
    nickname: string,
    phone: string,
    avatar_url: string | null,
    has_profile: boolean,
    profile: {
      major: string | null,
      grade: number | null,
      goal: '考研' | '考公' | '就业' | '出国' | '未定' | null
```

| 字段 | 类型 | 说明 |

| 字段 | 类型 | 说明 |

## 六、API接口

### 6.1 获取当前用户信息

```
GET /api/auth/me
```

**响应：**
```json
{
  "success": true,
  "data": {
    "id": 1,
    "nickname": "张三",
    "phone": "13800138001",
    "has_profile": true,
    "profile": {
      "major": "经济管理学院",
      "grade": 2,
      "goal": "考研"
    }
  }
}
```

### 6.2 更新学生画像

```
POST /api/auth/profile
```

**请求：**
```json
{
  "nickname": "张三",
  "major": "经济管理学院",
  "grade": 2,
  "goal": "考公"
}
```

**响应：**
```json
{
  "success": true,
  "message": "资料完善成功"
}
```
